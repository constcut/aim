<h2>Описание главного меню</h2><br>
<img src="mainMenu.png">
<pre>

Некоторые пункты меню скрываются при портретной ориентации экрана,
доступ к ним по прежнему можно получить через другие кнопки,
при долгом нажатии - из других кнопок появится меню.

1) назад - позволяет перейти к преидущему окну

2) новая таблатура

3) открыть таблатуру из файла

4) окно просмотра таблатуры

5) меню встроенных тестов - скрывается

6) настукивание ритма - скрывается

7) паттерн ввод
Содержит внутри меню:
	* настукивание
	* запись
	* морзе ввод
	* и сам пункт паттерн ввода

8) конфигурация
Содержит внутри меню: 
	* информация
	* тесты
	* и сам пункт конфигурации

9) меню записи (не отобржает запись на андройде)

10) ввод азбукой морзе

11) информация (меню about)

12) клавиша undo

13) проигрывание таблатуры

14) открытие панели

</pre><br>

<pre>

Существующие типы окон:

А) Окно таблатуры
Возможные действия:

*Выбор такта нажатием - откроется выбранная дорожка на заданном такте

*Листание таблатуры вправо влево - навигация по таблатуре

Б) Окно трека

*Выбор такта и бита при помощи нажатия

*Листание таблатуры вверх вниз

*Двойное нажатие выделяет конец области для копирования

В) Окно встроенных тестов

*Содержит предустановленные тесты для альфа тестирования, первые два раздела содержут базовые функции,
третий - композиции меня и товарищей

Г) Окно настукивания ритма

*Позволяет настукивать ритм и преобразовать его в таблатуру - все инструкции выполняются простыми кнопками

Д) Окно ввода паттерном

*Позволяет задать ударные паттерном

Е) Окно конфигурации

*Позволяет просмотреть загруженные конфигурационные параметры

*Содержит кнопки масштабирования

*Содержит кнопки влючения основных секций логирования

*Позволяет масштабировать экран жестом(на других окнах жест не активен)

Ж) Окно записи и распознавания
*Позволяет осуществить запись
	
*Отображает записанную звуковую волну по которой можно навигироваться прокрутой вправо и влево, и проводит спектральный анализ записи
	(пока не работает на андройде)

З) Окно Морзе ритма

*Позволяет ввести текст и генерирует из него ритмическую дорожку

И) Окно информации

*Содержит информацию о проекте

К) Стартовое окно

*Содержит справочную информацию которую вы только что прочли
</pre>

<h2>Логирование</h2>
<pre>
Существует система логировния - для того чтобы помочь находить баги или причины крашей.
Сейчас лог файл лежит в корне внутренней памяти вашего устройства:
Имя файлай: Guitarmy.log

Пожалуйста отметьте - каждый раз вы запускаете приложение - лог перезаписывается,
потому если вам необходимо сохранить лог после краша - сделайте это до очередного запуска.
</pre>
